# YouTube Sticky Player Concept

Created by Molham Stef

## Live Demo

https://stef675.github.io/youtube-sticky-player-concept/

## Problem

When users browse recommended videos on YouTube Desktop, the current video moves away with the page, interrupting the viewing experience.

## Proposed Solution

Keep the video player fixed and visible while allowing the recommendations section to scroll independently.

## Benefits

- Better user experience
- Continuous video watching
- Easier content discovery
- Potential increase in engagement and watch time

## Technologies

- HTML
- CSS
- JavaScript
- Figma

## About

This project started as a simple observation while using YouTube on desktop. I designed the idea in Figma and then implemented it using HTML, CSS and JavaScript to demonstrate how a sticky video player could improve the browsing experience while watching videos.
